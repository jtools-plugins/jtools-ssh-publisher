# SSH Publisher

IntelliJ IDEA SSH 客户端插件，支持 SSH 连接管理、终端操作、文件系统浏览和批量文件上传。

## 功能特性

### SSH 连接管理
- 密码和密钥两种认证方式
- 配置分组管理（group+name 唯一标识）
- 连接测试
- 心跳保活，防止超时断开

### SSH 终端
- 基于 JediTerm 终端模拟器
- 自动适配 IDEA 主题颜色
- 中文字体支持
- 多终端标签页
- Tab 拖拽排序
- Tab 右键分屏（向右/向下拆分）

### 文件系统 (SFTP)
- 远程文件浏览器
- 文件/文件夹上传下载（带进度显示）
- 新建文件夹、新建文件
- 重命名、删除
- 双击文件在 IDEA 编辑器中打开编辑
- 编辑保存后自动同步到远程服务器
- 支持 IDEA 本地历史记录
- 文件属性查看（大小、权限、修改时间）
- 快速搜索定位
- 懒加载目录（展开时加载）

### 文件上传
- 单服务器上传
- 批量上传（同一文件上传到多台服务器）
- 上传队列管理（进度、停止、重试、删除）
- 自定义远程文件名
- 前置/后置脚本执行
- 临时脚本支持

## 使用说明

### 添加 SSH 连接
1. 点击工具栏「+」按钮
2. 填写名称、主机、端口、用户名
3. 选择认证方式（密码/密钥）
4. 可选配置前置/后置脚本
5. 点击「测试连接」验证

### 打开终端
- 双击 SSH 配置

### 打开文件系统
- 右键 SSH 配置 → 文件系统

### 编辑远程文件
- 在文件系统中双击文件
- 文件下载到本地缓存并在 IDEA 编辑器中打开
- 保存时自动同步到远程服务器
- 支持 IDEA 本地历史记录（Local History）

### 上传文件
- 右键 SSH 配置 → 上传文件
- 或在文件系统面板点击上传按钮

### 批量上传
1. 点击工具栏「批量上传」
2. 选择本地文件
3. 勾选目标服务器
4. 点击服务器配置脚本
5. 添加到上传队列

### Tab 分屏
- 右键 Tab → 向右拆分 / 向下拆分
- 拖拽 Tab 可调整顺序
